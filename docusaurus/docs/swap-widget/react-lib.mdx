## Installation

### 1. yarn or npm install the library

```
yarn add @alpha-defi/raydium-widgets
```

### 2. Add widget to your UI

```
import { SwapWidget } from "@alpha-defi/raydium-widgets";
<SwapWidget></SwapWidget>
```
